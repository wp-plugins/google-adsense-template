== Description ==

Add a Google Adsense Template to use on your Wordpress pages.  Place Google Adsense ads on any page.


###Quick Start Guide

* [Google Adsense Template quick start guide at thinklandingpages.com] (http://www.thinklandingpages.com/google-adsense-template/)

###What you get when you use the Google Adsense Template plugin

*  Put Google Adsense ads on any page.
*  In content top Google Adsense ad
*  In content bottom Google Adsense ad
*  Setting page to save both, top and bottom, Google Adsense ads


###Add Google Adsense Template to any page

Whether you are writing a page about your company or another company, you can include Google Adsense ads with this page template

###Quick Start Guide

* [Google Adsense Template quick start guide at thinklandingpages.com] (http://www.thinklandingpages.com/google-adsense-template/)

###Unlimited Google Adsense Template Pages

Add as many Google Adsense Template Pages to your site as you like.  The Google Adsense Template plugin lets you create Google Adsense Template Pages and save them for when ever you are ready to use them.



== Installation ==


1. Upload `google-adsense-template` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click **Google Adsense Template** on the admin menu to enable and set your options.


== Frequently Asked Questions ==

= Do I have to know how to program or design? =

No.  The plugin does the programming and design.

= Is there a limit to the number of Google Adsense Template Pages I can create? =
You can create an unlimited number of Google Adsense Template Pages.

= Can the Google Adsense Template be used on my homepage? =
Yes, you would have to set a static page as your homepage and then set that page to use the Google Adsense Template.

= Can I use Adsense Template Ultimate on any post or page? =
Adsense Template Ultimate can only be used on Wordpress pages where you can select a template.
